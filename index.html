Cultsig

Open source tools for running a cultural event listing in your area.

A simple, no-frills platform for art openings, DIY shows, film screenings, theater, jazz nights, and artist talks etc.

Craigslist for Culture.

What It Is

Cultsig gives you everything you need to run a community event calendar: submission forms, moderation tools, a weekly newsletter, and a clean public listing. No algorithm. No Paid Features. Chronological order, always.

What You Get

  • Public event listing with submission form
  • Moderation queue and duplicate detection
  • Weekly newsletter with subscriber preferences
  • Designed to run cheaply on standard cloud hosting

How To Use It

Fork the repo, configure it for your city, and deploy. Documentation and setup guides are in progress. The goal is to get a new region running in an afternoon.

Code will be at github.com — coming soon.
